# 07 - Validation

## Objective

Validate that Grafana accepts the Microsoft Entra ID SAML configuration.

## Validation Performed

- Grafana SAML wizard completed
- Microsoft Entra SAML values configured
- IdP metadata URL added to Grafana
- User mapping configured
- SAML status enabled

## Result

Grafana accepted the SAML provider settings and displayed the SAML configuration as enabled.
